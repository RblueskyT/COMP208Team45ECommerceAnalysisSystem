Introduction of E-Commerce Analysis System

① This system aims to provide an platform for trading companies which can help conduct commodity management, data mining on historical sales and intelligently provide suggestions about future marketing strategies.
② This system is a WEB Application. Staff in the trading company which are in different positions have different authorities to interact with the system. For example, sales manager can get access to add or update the information of commodities , but data analyst has no this permission.
③ If you have further questions, please Click "FAQ" which is under "Contact Us" to find answers.
